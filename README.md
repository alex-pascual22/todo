# Todo Management API
A simple RESTful Todo Management application built using Spring Boot.

## Technologies
- Java 21
- Spring Boot
- Spring Data JPA
- H2 Database
- Lombok
- Maven

## Features
- Create Todo
- View Todo/s
- Update Todo
- Delete Todo

## Project Structure
- Controller
- Service
- Repository
- DTO
- Mapper
- Entity

## API Testing
Endpoints tested using Postman.
